---
title: Generate a counterfactual deposit address
description: >
  Returns a counterfactual deposit address that the user can send funds to in
  order to initiate a crosschain transfer. This address is deterministically
  computed based on the swap parameters. The user sends the input token to this
  address on the origin chain, and Across handles the crosschain transfer to the
  recipient on the destination chain.


  The response includes the deposit address, token details, and an optional swap
  transaction that must be executed before sending funds (e.g. for token
  approvals or wrapping).

  Note: Arbitrum is the only supported origin chain and HyperEVM/HyperCore are
  the supported destination chains for now.
