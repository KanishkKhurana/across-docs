The Across API provides programmatic access to cross-chain bridging and swap functionality. Use these endpoints to get quotes, check limits, query deposit status, and more.

## Server URLs

| Environment | Base URL |
|-------------|----------|
| Production  | `https://app.across.to/api` |
| Testnet     | `https://testnet.across.to/api` |

## Authentication

All endpoints are publicly accessible. No API key is required. Production integrations should register for an [integrator ID](https://docs.across.to) (a 2-byte hex identifier) and pass it as a query parameter.
