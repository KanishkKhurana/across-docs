The Across API provides programmatic access to cross-chain bridging and swap functionality. Use these endpoints to get quotes, check limits, query deposit status, and more.

## Server URLs

| Environment | Base URL |
|-------------|----------|
| Production  | `https://app.across.to/api` |


## Authentication

All API requests require a Bearer token passed via the `Authorization` header.

<Callout type="info">
To get your API key, reach out to us on [Telegram](https://t.me/acrosstg).
</Callout>
